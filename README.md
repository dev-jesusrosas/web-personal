## 📋 Descripción
 
Portafolio profesional diseñado para presentar mis habilidades como **Analista de Datos** e **Ingeniero de Sistemas en formación**. El sitio está construido íntegramente con HTML5 y CSS3 sin librerías externas, priorizando rendimiento, accesibilidad y un diseño editorial moderno.
 
---
 
## ✨ Características
 
- **Sin dependencias** — HTML y CSS puros, sin frameworks ni librerías de UI
- **Diseño responsivo** — Adaptado para móvil, tablet y escritorio
- **Modo oscuro/claro automático** — Detecta la preferencia del sistema (`prefers-color-scheme`)
- **Animaciones CSS** — Transiciones suaves y efectos de scroll con `IntersectionObserver`
- **Navbar sticky** con efecto glassmorphism al hacer scroll
- **Menú hamburguesa** funcional para dispositivos móviles
- **Accesibilidad** — Atributos `aria-label`, enlaces con `rel="noopener"`, estructura semántica

## 🚀 Uso Local
 
No requiere instalación ni servidor. Basta con clonar y abrir el archivo:
 
```bash
git clone https://github.com/dev-jesusrosas/Web-portafolio-Html-Css.git
cd Web-portafolio-Html-Css
# Abrir index.html en tu navegador preferido
```
 
O con VS Code usando la extensión **Live Server**:
 
```
Click derecho en index.html → "Open with Live Server"
```

## 🎨 Decisiones de Diseño
 
- **Tipografía:** [Syne](https://fonts.google.com/specimen/Syne) (display) + [DM Sans](https://fonts.google.com/specimen/DM+Sans) (cuerpo)
- **Paleta:** Tema oscuro `#080c14` con acento azul `#3b82f6` y soporte de modo claro
- **Layout:** CSS Grid y Flexbox nativos, sin Bootstrap ni similares
- **Animaciones:** CSS `@keyframes` + `IntersectionObserver` para reveals al hacer scroll
